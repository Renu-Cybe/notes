## 分析原则

1. **用自己的话重写** — 不是复制粘贴
2. **提取可借鉴的设计** — 而非关注功能
3. **连接现有知识** — 与我当前的系统有什么关系？
4. **评估采纳价值** — 是否值得投入时间？

---

## 决策框架

新项目评估 → 三问：
1. **信任**：项目是否成熟、活跃？
2. **复利**：能积累什么能力？
3. **杠杆**：能放大什么产出？

三✅ 才采纳。
