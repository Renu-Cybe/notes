# Fooocus 项目分析

> 更新时间：2026-03-31

## 项目概览

| 维度 | 内容 |
|------|------|
| **Star** | 47,962 |
| **简介** | 离线、开源、免费的图像生成软件，类 Midjourney 体验 |
| **技术栈** | Python + Gradio + Stable Diffusion XL |
| **状态** | LTS（有限长期支持），仅修复 bug |

## 核心能力

| 能力 | 说明 |
|------|------|
| **文生图** | 内置 GPT-2 提示词处理引擎，自动优化提示词 |
| **图像变化/放大** | Subtle/Strong 变化、1.5x/2x 放大 |
| **Inpaint/Outpaint** | 自研 inpaint 算法 |
| **风格预设** | 多种风格一键切换 |

## 借鉴点

| 点 | 启发 |
|----|------|
| **简化用户体验** | 隐藏复杂参数，让用户专注于提示词 |
| **内置优化** | 自动优化提示词，降低使用门槛 |
| **离线优先** | 隐私可控，无需云端 |

|--------|------|
