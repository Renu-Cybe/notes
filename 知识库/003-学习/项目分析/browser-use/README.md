# browser-use 项目分析

> 更新时间：2026-03-31

## 项目概览

| 维度 | 内容 |
|------|------|
| **Star** | 85,241 |
| **简介** | AI 驱动的浏览器自动化框架，自然语言描述任务即可自动执行 |
| **技术栈** | Python + Playwright |
| **活跃度** | 今日更新（2026-03-31）|

## 核心能力

| 能力 | 说明 |
|------|------|
| **自然语言驱动** | 无需写脚本，描述任务即可 |
| **AI 自适应** | 自动判断导航策略，适应不同网站 |
| **多模型支持** | Browser Use Cloud、Claude、Gemini 等 |
| **云端 stealth** | 可用云端浏览器，反爬友好 |

## 借鉴点

| 点 | 启发 |
|----|------|
| **自然语言接口** | 用 LLM 作为浏览器操作的"翻译层"，降低自动化门槛 |
| **任务而非步骤** | 用户描述目标，AI 自动规划步骤 |
| **多站点适应** | 同一套代码适应不同网站结构 |

## 与本项目的关系

**已融合到 web-access 技能**（v2.5.0）：

| 模式 | 用途 |
|------|------|
| CDP | 登录态继承、精确控制 |
| browser-use | 批量调研、自然语言任务 |
