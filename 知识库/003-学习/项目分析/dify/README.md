# Dify 项目分析

> 更新时间：2026-03-31

## 项目概览

| 维度 | 内容 |
|------|------|
| **Star** | 135,174 |
| **简介** | 开源 LLM 应用开发平台，可视化编排 AI 工作流 |
| **技术栈** | TypeScript + Python + Docker |
| **活跃度** | 今日更新（Linux Foundation 托管）|

## 核心能力

| 能力 | 说明 |
|------|------|
| **Workflow** | 可视化画布构建和测试 AI 工作流 |
| **模型支持** | 无缝集成数百个 LLM（GPT、Claude、Llama 等）|
| **RAG Pipeline** | 完整的文档摄取到检索能力 |
| **Agent 能力** | 基于 Function Calling/ReAct，内置 50+ 工具 |
| **LLMOps** | 应用日志监控、性能分析 |

## 借鉴点

| 点 | 启发 |
|----|------|
| **可视化编排** | 降低 AI 工作流搭建门槛 |
| **模型无关** | 一套系统适配多种 LLM |
| **从原型到生产** | 快速迭代 + 生产部署一体化 |

## 与本项目的关系

| 场景         | 应用               |
| ---------- | ---------------- |
| **技能编排平台** | 可用 Dify 构建技能调度系统 |
| **企业内部工具** | 快速搭建 AI 辅助工具     |
| **工作流可视化** | 将 Temper 技能体系可视化 |
