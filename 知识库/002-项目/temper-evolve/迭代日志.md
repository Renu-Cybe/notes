## 核心架构

```
temper/
├── core/
│   ├── errors.py          # 结构化错误系统（学习 Codong）
│   └── types.py           # 类型验证
└── tools/
    ├── fs.py              # 文件系统模块
    ├── shell.py           # 命令执行模块
    └── self.py            # 自我进化工具（Day 3 新增）
```

---

## 工具清单

| 模块 | 工具 | 功能 |
|------|------|------|
| **FS** | fs.read/write/edit/exists/list/read_json | 文件操作 |
| **Shell** | shell.run | 命令执行 |
| **Self** | self.snapshot/diff/log/list_snapshots | 自我进化记录 |
| **Chain** | chain / parallel | 工具链组合（Day 4）|
