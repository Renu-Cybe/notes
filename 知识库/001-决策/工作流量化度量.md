# 工作流量化度量体系

**决策日期**: 2026-03-31
**决策类型**: CTO-技术决策

## 核心原则

> 不能度量就无法优化，不能优化就无法复利。

## 四维指标体系

### 1. 效率指标（时间）
- 任务耗时：人工耗时 - Agent耗时（目标：降低80%）
- 并行度：并发技能数（目标：≥3个）
- 响应时间：需求→首版（目标：<30分钟）

### 2. 质量指标（输出）
- 一次通过率：无需修改直接交付的比例（目标：≥70%）
- 修改轮次：初稿到终稿迭代次数（目标：<2轮）
- 合规率：格式/内容检查通过率（目标：100%）

### 3. 复利指标（积累）
- 知识库增长率：月新增条目数（目标：+10条/月）
- 技能复用率：调用次数/周（持续提升）
- 自动化覆盖率：自动步骤占比（目标：≥90%）

### 4. 业务指标（结果）
以公众号为例：
- 输入：热点匹配度≥80%、素材真实性100%
- 过程：成文速度<30分钟、排版合规率100%
- 结果：周发布量≥2篇、关注转化率>0%

## ROI 计算

```
工作流价值 = (节省人工时间 × 时薪) - (维护成本 + 配置成本)
复利价值 = 知识库资产 + 技能复用节省总时间
```

## 实施要点

1. 新建工作流时同时定义度量指标
2. 执行时自动/手动记录数据
3. 复盘时用数据驱动优化
4. 优化后对比度量验证效果
